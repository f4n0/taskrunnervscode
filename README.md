# Task Runner

This extension adds a "Task Runner" view to the VS Code Explorer pane, allowing you to visualize and individually run auto-detected tasks in your project. It supports task runners such as Gulp, Grunt, Jake, and npm, making it easier to manage and execute project automation tasks directly from the editor.

## Features

- Adds a dedicated Task Runner view in the Explorer pane
- Lists and runs auto-detected tasks (npm, Gulp, Grunt, Jake)
- Simple interface for executing tasks without leaving VS Code
