# Change Log
All notable changes to the "taskrunnercode" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.0.1]
### Added
- Tasks can now be grouped by source (Workspace, User, package.json) in a tree view
- New setting `taskRunner.groupBySource` to enable/disable grouped view (disabled by default)
